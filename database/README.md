# Database

This folder contains the initial PostgreSQL schema for ProcessMind Assistant.

## Files

- `schema.sql` → initial database schema
- future migrations can be added later

## Stack

- PostgreSQL
- pgvector

## Main features supported

- organizations and users
- uploaded documents
- document chunk indexing
- semantic search
- chat sessions
- AI query logs
