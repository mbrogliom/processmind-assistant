# Backend Architecture

The backend of ProcessMind Assistant is built using FastAPI and follows a modular architecture.

The goal of this structure is to keep the system maintainable, scalable and easy to extend.

The backend is divided into several modules, each responsible for a specific domain of the system.

## Main backend modules

- authentication
- users
- organizations
- documents
- document processing
- embeddings
- retrieval
- chat
- logging
- administration

Each module exposes API endpoints and internal services.

# Architettura del backend

Il backend di ProcessMind Assistant è costruito utilizzando FastAPI e segue un'architettura modulare.

L'obiettivo di questa struttura è mantenere il sistema manutenibile, scalabile e facile da estendere.

Il backend è suddiviso in diversi moduli, ognuno responsabile di uno specifico dominio del sistema.

## Moduli principali del backend

- autenticazione
- utenti
- organizzazioni
- documenti
- elaborazione documenti
- embeddings
- retrieval semantico
- chat AI
- logging
- amministrazione

Ogni modulo espone API e servizi interni.
