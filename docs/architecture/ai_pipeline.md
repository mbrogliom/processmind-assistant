# AI Pipeline

The AI pipeline transforms documents into searchable knowledge.

Steps:

1 Document upload  
2 Text extraction  
3 Text chunking  
4 Embedding generation  
5 Vector indexing  
6 Semantic search  
7 Context construction  
8 LLM response generation
